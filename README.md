## Github Subtree
- 여러 개의 Git Repository를 폴더 하나(큰 Repository)에서 관리할 수 있게 하는 기능
  - 정확히는 Repository가 큰 Repository의 하위 파일로 분류되어 들어감
- History를 그대로 유지하게 함으로, Commit 내역 및 잔디를 보존할 수 있음
